Named-Data Network (NDN)
========================
provide elegant and simple ndn library for research and testing purpose

What is it?
===========
1. TLV (tlv.go)
2. Raw NDN TLV encoder/decoder (encoding.go)
3. NDN packet (ndn.go)
4. NDN server (on top of tcp)

Import
======
```
import "github.com/taylorchu/ndn"

```
